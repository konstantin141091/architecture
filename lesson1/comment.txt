Текстовое описание папки src
Структура устроена по патерну mvc.
Контролер получает запрос(request). Передает эти данные в сервис.
В сервисе основная бизнес логика.
Сервис получает данные о сущности из хранилища(repository).
В repository содержаться методы для работы с хранилещем(в данном проекте хранилище тамже в виде массива)
Также есть сама сущность (entiy или модель), которая содержит гетыры.
